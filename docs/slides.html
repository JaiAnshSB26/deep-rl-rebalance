Slide 17: Q&A Preparation

Anticipated Questions & Answers

Q: Why not just use equal-weight? > A: Equal-weight worked well in this specific test period (2022-2025), but RL adapts to changing regimes. In different market conditions (e.g., 2008 crisis), dynamic strategies can outperform. We‚Äôre trading flexibility for cost.

Q: How would you deploy this in production? > A: Wrap model in Flask API, monitor for distribution drift, implement circuit breakers if Sharpe drops below threshold. Use Docker + Kubernetes for scaling. Add human-in-the-loop for regime shifts.

Q: What about overfitting? > A: We used validation set for model selection, cross-sectional normalization to prevent time-series overfitting, and tested on 4 years of unseen data. Still, regime changes (2012-2018 vs 2022-2025) are a challenge.

Q: How does this compare to BlackRock/AQR strategies? > A: They use factor models + optimization. We use direct policy learning. Ours is more adaptive but higher variance. Institutional funds might ensemble both approaches.

Backup 1: PPO Algorithm Details

Proximal Policy Optimization

Objective: L^{\text{CLIP}}(\theta) = \mathbb{E}_t \left[ \min(r_t(\theta) \hat{A}_t, \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) \hat{A}_t) \right]

Where: - r_t(\theta) = \frac{\pi_\theta(a_t|s_t)}{\pi_{\theta_{\text{old}}}(a_t|s_t)} (probability ratio) - \hat{A}_t = GAE advantage estimate - \epsilon = 0.2 (clip range)

Why PPO? - Stable (clipped objective prevents large updates) - Sample-efficient (on-policy with multiple epochs) - State-of-the-art for continuous control

[Visual: PPO objective function graph showing clipping]

Backup 3: Baseline Strategy Details

5 Classic Strategies

1. Equal Weight Buy & Hold: - w_i = 1/N, no rebalancing

2. Periodic Rebalancing: - Rebalance to 1/N monthly (first trading day)

3. Risk Parity: - w_i \propto 1/\sigma_i (inverse volatility)

4. Momentum Tilt: - w_i = 1/N \cdot (1 + \alpha \cdot \text{mom}_i)

5. Mean-Variance: - \arg\min_w \frac{1}{2} w^\top \Sigma w - \lambda \mu^\top w - Ledoit-Wolf shrinkage covariance

[Visual: Table showing baseline formulas]

Backup 4: Statistical Test Details

Diebold-Mariano Test

Null Hypothesis: E[d_t] = 0 where d_t = L(e_t^{\text{RL}}) - L(e_t^{\text{baseline}})

Test Statistic: \text{DM} = \frac{\overline{d}}{\sqrt{\text{Var}(\overline{d})}}

Implementation: - Use Newey-West HAC standard errors (lag = T^{1/3}) - Two-tailed test - Null: Equal predictive accuracy

Result for RL vs EW: - DM = 1.183 - p-value = 0.237 - Cannot reject null at 5% level

[Visual: DM distribution with rejection regions]
